# TasksApp - Frontend (Android)

## Описание
**TasksApp** — это мобильное Android-приложение для управления задачами. Пользователи могут создавать, редактировать и удалять задачи, сортировать их по категориям и приоритету, а также просматривать задачи на календаре. Приложение поддерживает учет статуса выполнения задач и предоставляет удобный интерфейс для быстрого взаимодействия.

## Функциональность
- Просмотр задач по датам через календарь (`CalendarView`).
- Фильтрация задач по категориям и приоритетам.
- Добавление, редактирование и удаление задач.
- Работа с категориями: создание, изменение, удаление.
- Использование **BottomNavigationView** для переключения между разделами:
    - Задачи
    - Календарь
    - Профиль
    - Настройки
- Поддержка адаптивного интерфейса с учетом статус-бара (`WindowInsetsCompat`).
- Асинхронная работа с API через `Retrofit2` и `CoroutineScope`.

## Технологии и библиотеки
- **Язык:** Kotlin
- **UI:** Android Jetpack, Material Components (TextInputLayout, Chips, FloatingActionButton, MaterialCardView)
- **Асинхронность:** Kotlin Coroutines (`lifecycleScope`)
- **Сеть:** Retrofit2 + Gson для работы с REST API
- **Архитектура:** MVVM-подобная структура (UI + API + DTO)
- **Совместимость:** Android API 21+


## Настройка и запуск
1. Клонируйте репозиторий и откройте проект в **Android Studio**.
2. Убедитесь, что установлены:
    - Android SDK API 21+
    - Gradle 7+
    - Подключение к интернету для Retrofit
3. Подключите устройство или эмулятор Android.
4. Постройте проект (`Build > Make Project`) и запустите (`Run > Run 'app'`).

## Важные моменты реализации
- **Цвет категорий и состояние задач** задаются через `ColorStateList` и HEX-коды.
- **Асинхронная загрузка задач и категорий** выполняется через `lifecycleScope.launch { ... }`.
- **Адаптация под статус-бар** реализована с помощью `ViewCompat.setOnApplyWindowInsetsListener`.
- **UI элементы:**
    - `CalendarView` для выбора даты.
    - `MaterialCardView` для карточек задач.
    - `ChipGroup` для фильтрации по категориям.
    - `FloatingActionButton` для добавления задач.
    - `BottomNavigationView` для навигации по основным разделам.
